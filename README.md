# 🏏 Sportz Realtime - Live Match Commentary

A real-time live match commentary application that delivers instant updates using WebSocket technology. This project ensures low-latency communication between server and clients, making it ideal for live sports tracking.

## 🚀 Features
- ⚡ Real-time commentary updates
- 🔄 Instant message broadcasting using WebSockets
- 👥 Multiple users can receive updates simultaneously
- 🌐 Scalable backend architecture
- 📡 Efficient client-server communication

## 🛠️ Tech Stack
- Node.js
- WebSocket (ws)
- JavaScript

## 💡 How It Works
- Server establishes a WebSocket connection with clients
- When a new commentary message is received, it is broadcast to all connected users in real-time
- Ensures fast and seamless live updates without page refresh

## 📌 Use Cases
- Live sports commentary
- Real-time chat applications
- Event tracking systems
